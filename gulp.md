易于使用：
通过代码优于配置的策略，Gulp让简单的任务简单，复杂的任务可管理

构建快速
利用Node.JS流的威力，你可以快速构建项目并减少频繁的ID操作

插件高质
Gulp严格的插件指南确保插件如你所期待的那样简洁高质的工作

npm install gulp --save-dev  //作为项目开发依赖安装
